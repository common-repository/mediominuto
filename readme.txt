MedioMinuto Wordpress Plugin API allows you to integrate your website in wordpress MedioMinuto to record and play videos on your blog easy way ..

== Description ==

MedioMinuto is a leader in providing video recording technology for the web. Without prior knowledge you can offer video on your blog with a few clicks. 

MedioMinuto plugin for Wordpress allows you to use this technology on your blog with just install the plugin and indicate your API key

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `mediominuto` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add your API Key on MedioMinuto preferences page.
1. When publish a new post, click on Grabar button and enjoy!
